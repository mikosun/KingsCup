# キングスカップ 🎲

飲み会ゲーム「キングスカップ」をWebブラウザで楽しめるアプリケーションです。

実際のトランプでやるのと同じように、ランダムにカードを引いて、対応するルールに従います。引いたカードはリセットするまで二度と出ません。

## 機能

- 🃏 **カードシャッフル** - 52枚のトランプ＋ジョーカー2枚をランダムに引く
- 🎯 **ルール表示** - 引いたカードに対応するルールを大きく表示
- ✨ **ハイライト機能** - 引いたカードのルールが右側で自動ハイライト
- 🔄 **リセット機能** - 全カードをリセットして最初からやり直し
- 📊 **統計表示** - 残りのカード枚数を表示
- 📱 **レスポンシブ** - スマートフォンでも楽しめる

## 使い方

1. `index.html` をWebブラウザで開く
2. 「カードを引く」ボタンをクリック
3. ランダムに引いたカードと対応するルールが表示される
4. 全カードを引き終わったら「リセット」ボタンで新しいゲーム開始

## ゲームルール

| カード | ルール |
|--------|--------|
| **A (1)** | 引いた人以外がグイ |
| **2** | 指名された人がグイ |
| **3** | 引いた人がグイ |
| **4** | 女子がグイ |
| **5** | 引いた人の左右がグイ |
| **6** | 男子がグイ |
| **7** | 出たら片手を上げて遅い人がグイ |
| **8** | 連帯責任の相棒を決める（次出たら前のは解消） |
| **9** | 好きなゲーム |
| **10** | 次のターン3枚引く |
| **J (11)** | 好きなルール作る |
| **Q (12)** | クエスチョンマスター（次出たら前のは解消） |
| **K (13)** | 4枚目を引いた人がショットをグイ |
| **JOKER** | 2枚目引いたらショット2杯 |

## インストール

このプロジェクトは静的ファイルのみです。サーバーは不要で、ファイルをダウンロードしてブラウザで開くだけで使用できます。

### ローカルで実行

```bash
# リポジトリをクローン
git clone https://github.com/yourusername/KingsCup.git
cd KingsCup

# ブラウザで開く
# index.html をダブルクリック、または
# 簡易サーバーで実行
python -m http.server 8000
# http://localhost:8000 にアクセス
```

## ゲームのコツ

- 📢 **大人数でプレイ** - 3～8人程度が最適です
- 🎉 **ルール8, 12は重要** - 連帯責任とクエスチョンマスターの関係を理解しましょう
- 🃏 **ジョーカーに注意** - 1回目と2回目で異なるペナルティ
- 💧 **飲み物を用意** - ゲームの上手さより運が重要です！
